# Calculadora con Validación de División por Cero

### 📝 Descripción
Calculadora interactiva en Python que realiza operaciones aritméticas básicas. El proyecto destaca por la implementación de controles de seguridad que detectan y previenen el error crítico de división por cero (ZeroDivisionError), garantizando la estabilidad del programa.

### 🚀 Tecnologías utilizadas
* **Lenguaje:** Python 3
* **Conceptos:** Manejo de excepciones específicas, validación preventiva de datos y operadores matemáticos.
